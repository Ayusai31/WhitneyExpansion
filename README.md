# 秘密基地怎么触发
- 在放学后触发**紫色字体**的“你看见惠特尼在大门口附近晃悠”后，就可以触发秘密基地事件。
- 出门找惠特尼，如触发**选项中有问他/她怎么了**的剧情，选择“问他/她怎么了”就能触发秘密基地了。
- 如果没有触发上一条的剧情，可以选择回溯回出门前，进学校并开关储物柜即可刷新门口事件。
